Commandes Description Usage Shortcut
ban Pour bannir un membre du serveur. &ban [@membre]
&ban [@member] [nombre de jour]
&ban [@member] [raison]
&ban [@member] [nombre de jour] [raison]
checkWarn Pour voir quels membres ont reçu des avertissements et combien. &checkWarn &cw
clear Pour supprimer un certain nombre de messages dans un channel. &clear
&clear [nombre de messages]
kick Pour exclure un membre du serveur. &kick [@member]
&kick [@member] [raison]
mute Pour muter un membre (ainsi il ne pourras plus envoyer de messages ou parler dans un vocal). &mute [@member]
&mute [@member] [temps en heure]		 &m
unban Pour débannir un membre du serveur. &unban [id]
&unban [id] [raison]
unmute Pour démuter un membre &unmute [@member]
unwarn Pour enlever un avertissement à un membre. &unwarn [@member]
&unwarn [@member] [raison]		 &uw
warn Pour donner un avertissement à un membre, au bout d'un certain nombre il sera banni (définissable avec &setWarn, par défaut 3). &warn [@member]
&warn [@member] [raison]		 &w
addRole Pour ajouter un rôle à un membre. &addRole [@role] [@member] &addR
autorole Pour ajouter un rôle automatiquement à chaque arrivant. &autorole [@role]
createRole Pour créer un rôle. (à la place de séparé ? tu dois mettre soit true (= oui) soit false (= non)) &createRole [nom]
&createRole [séparé ?] [nom]		 &createR
deleteRole Pour supprimer un rôle. &deleteRole [@role]
&deleteRole [nom du role]		 &deleteR
reactionRole Pour créer un message réaction-rôle. (réaction-rôle = clique sur une réaction pour recevoir un rôle) &reactionRole
removeRole Pour enlever un rôle à un membre. &removeRole [@role] [@member] &removeR
blockCommand Pour bloquer une commande de ce bot dans ce serveur. &blockCommand [nom de la commande] &bc
lock Enlever le droit d'écriture aux membres dans un channel. &lock
poll Pour faire des sondages. &poll [question / message]
setChannel Pour activer un des channels suivants:
join: messages de bienvenue pour les membres qui rejoignent le serveur
leave: messages d'au revoir pour les membres qui quittent le serveur
logs: les logs du serveur
Pour en activer un, il faut envoyer dans le channel souhaité la commande avec here
Et pour en désactiver un, il faut envoyer dans un channel la commande avec stop		 &setChannel [channel à activer] [here / stop]
Exemple: &setChannel join here		 &setc
setGuildLanguage Pour changer la langue du serveur utilisé par le bot. &setGuildLanguage [fr / en] &setGuildl
setPrefix Pour changer le prefixe du bot dans ce server. &setPrefix [nouveau prefix]
setWarn Pour définir le nombre d'avertissements qu'un membre doit recevoir pour être banni. &setWarn [nombre d'avertissement]
unblockCommand Pour débloquer une commande de ce bot dans ce serveur. &unblockCommand [nom de la commande] &ubc
unlock Redonner le droit d'écriture aux membres dans un channel. &unlock
gmn Il pense à un nombre et tu dois le trouver. &gmn
hangman Pour jouer au jeu du pendu. &hangman &hg
randomSentence Envoie une phrase totalement aléatoire (et sûrement dépourvue de sens). &randomSentence &rs
tictactoe Pour jouer au morpion (si tu ne mentionne personne, tu joueras contre le bot) &tictactoe &morpion
weirdWord Il mélange un mot et tu dois retrouver le mot dans un temps limité. Pour jouer à plusieurs, faites weirdWord multi. &weirdWord
&weirdWord multi		 &ww
avatar Envoie en grand votre avatar ou celui de la personne mentionnée. &avatar
&avatar @user		 &pp
botInfo Toutes les informations sur &Stunning. &botInfo &bot
bug Pour prévenir le modérateur du bot d'un bug ou d'un problème concernant le bot. &bug [problème]
help Pour avoir une liste de toute les commandes du bot (si tu précise le nom d'une commande, alors il t'enverra seulement les détail de celle-ci). &help
&help [commande]		 &h
invite Envoie un lien pour m'inviter sur un serveur. &invite
postIt Pour récupérer le texte que tu as enregistré avec la commande setPostIt &postIt [texte / message]
servInfo Toutes les informations sur le serveur sur lequel vous avez envoyez la commande. &servInfo &serv
setPostIt Pour enregister un texte (et le récupérer avec postIt). &setPostIt
setUserLanguage Pour changer la langue du bot pour toi. &setUserLanguage [fr / en] &setUserl
suggestion Pour transmettre au modérateur du bot une suggestion. &suggestion [votre suggestion]
userInfo Toutes les informations sur la personne mentionnée ou sur toi si personne ne l'est. &userInfo
&userInfo [@user]		 user
vote Envoie le lien pour voter pour moi. &vote
